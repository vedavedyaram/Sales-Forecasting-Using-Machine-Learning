-> Sales Forecasting Using Machine Learning

-> Project Overview
This project predicts future sales using historical sales data from the Sample Superstore dataset.

-> Dataset
- Source: Kaggle Sample Superstore Dataset
- Records: 9,994
- Features: 21+

-> Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

-> Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Business Insights

-> Models Used
- Linear Regression
- Random Forest Regressor

-> Evaluation Metrics
- RMSE
- MAPE
- R² Score

-> Business Insights
- Technology category generated the highest sales.
- Seasonal sales trends were identified.
- Discounts influenced profitability.
